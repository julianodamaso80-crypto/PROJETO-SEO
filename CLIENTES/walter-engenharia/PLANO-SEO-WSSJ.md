## 5. SEO TECNICO — CHECKLIST

### 5.1 Setup Inicial
- [ ] Registrar dominio (sugestoes: wssjengenharia.com.br ou wssjservicos.com.br)
- [ ] Hospedagem WordPress otimizada (Hostinger Business ou Cloudways)
- [ ] Instalar WordPress com tema otimizado para SEO (Flavor: developer) como flavor flavor flavor flavor ??
- [ ] SSL (HTTPS) ativo
- [ ] Instalar plugins essenciais:
  - **Rank Math SEO** (otimizacao on-page, schema, sitemap)
  - **WP Rocket** (cache e performance)
  - **Imagify** (compressao de imagens)
  - **WPForms** (formularios de contato/orcamento)
  - **Insert Headers and Footers** (GTM, pixels)

### 5.2 Otimizacao On-Page (cada pagina)
- [ ] Title tag com keyword principal + localizacao + marca
- [ ] Meta description com CTA e keyword
- [ ] H1 unico com keyword
- [ ] H2/H3 com keywords secundarias
- [ ] URL slug limpo e descritivo
- [ ] Imagens com alt text descritivo
- [ ] Internal linking entre paginas de servico, regiao e blog
- [ ] Schema markup: LocalBusiness, Service, FAQPage, BreadcrumbList

### 5.3 Core Web Vitals
- [ ] LCP < 2.5s (imagens otimizadas, cache)
- [ ] FID/INP < 200ms (JS minimo)
- [ ] CLS < 0.1 (dimensoes de imagem declaradas)

### 5.4 Indexacao
- [ ] Sitemap XML gerado (Rank Math)
- [ ] robots.txt configurado
- [ ] Google Search Console configurado e sitemap enviado
- [ ] Bing Webmaster Tools configurado
- [ ] Google Analytics 4 instalado
- [ ] GTM instalado (para futuros eventos de conversao)
